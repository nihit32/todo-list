A simple and responsive To-Do List application built with HTML, CSS, and JavaScript.

✨ Features:

* Add new tasks
* Edit existing tasks
* Delete tasks
* Input validation to prevent empty tasks
* Dynamic DOM manipulation using createElement()
* Clean and user-friendly interface

🛠️ Technologies Used:

* HTML5
* CSS3
* JavaScript (DOM Manipulation & Event Handling)

This project was built to practice JavaScript fundamentals, DOM manipulation, event handling, and CRUD operations without using any external libraries or frameworks.
